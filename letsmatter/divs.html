Divs, as in Divisors

Previously, we learned how to style different elements by selecting them by name. We also learned how to change elements without changing other elements that share the same tag.

Sometimes, we want to group elements together. We want to do this for the following reasons:

  • creating a different visual aesthetic for a section of the page
  • associate elements with eachother regardless of tag
  • and more (i couldn't think of anything else lol, feel free to add on in the discussion XD )

We do this by:

  1. creating a <div> tag
  2. giving the div tag a name
  3. selecting the div, and applying styling to it.
