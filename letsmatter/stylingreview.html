Styling Review

In the last article, we learned how to style HTML elements by selecting them in the <style> tag.

We do this using the following syntax: 

      
    [ELEMENT]
    {
      [PROPERTY] : [VALUE];
    }
  • We first write the name of the ELEMENT, following curly braces. Inside those curly braces, we write a statement that change how this element is displayed.
  • Each statement starts with the PROPERTY name, which is what we want to change about the element. (i.e., color, font-size, font, etc)
  • Then we write a colon.
  • After the colon, we write the VALUE, or how we want to change the element. (i.e., red, Helvetica, 3em, etc.)
