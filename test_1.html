public int fibonacci(int n)
{
if(n == 0) return 0; else if(n == 1) return 1;
else return fibonacci(n - 1) + fibonacci(n - 2);
}

method call is the function itself, it is called again in the recursive step
the base cases are terms '0' and '1' which return 1
this is the recursive step, here it calls sums the value returned from other calls with a modified parameter
